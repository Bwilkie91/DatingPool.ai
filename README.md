# DatingPool

Event-based dating app coming Summer 2026 for iOS and Android.

## About

DatingPool is a revolutionary dating app that connects people through shared experiences. Instead of swiping through profiles, users discover and connect at events they're both attending - concerts, workshops, sports events, and more.

## Features

- **Event-Based Matching**: Connect with people attending the same events
- **Shared Interests**: Built-in conversation starters through common activities
- **Natural Meetings**: Meet in person at events you both love
- **Safe & Secure**: Verified profiles and event-based meetups

## Getting Started

### Prerequisites

- Node.js 16+ and npm

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```
